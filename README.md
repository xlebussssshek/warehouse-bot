# Telegram-бот для складского учёта

## Описание проекта
Бот для управления остатками товаров на складе. 
Позволяет добавлять, списывать товары и выгружать отчёты в Excel.

## Стек технологий
- Python 3.12
- Aiogram 3.x
- SQLite
- OpenPyXL
- Logging

## Функционал
- Проверка остатка по артикулу
- Добавление товаров (поддержка названий любой длины!)
- Безопасное списание (запрет ухода в минус)
- Выгрузка Excel-отчёта
- Белый список пользователей
- Логирование всех действий

## Быстрый старт

1. Клонируйте репозиторий
    git clone https://github.com/xlebussssshek/warehouse-bot.git
    cd warehouse-bot
2. Создайте виртуальное окружение
    python -m venv .venv
    .venv\Scripts\activate  # Windows
3. Установите зависимости
    pip install -r requirements.txt
4. Создайте файл config.py на основе config.example.py
5. Вставьте ваш токен и Telegram ID
6. Запустите бота
    python main.py


Пример работы
/add A-001 Мышь оптическая беспроводная 10
Добавлено 10 шт.
A-001 - Мышь оптическая беспроводная
Текущий остаток: 15 шт.

/stock A-001
A-001 - Мышь оптическая беспроводная
Остаток: 15 шт.

/report
Отчёт: остатки_2026-02-12.xlsx
